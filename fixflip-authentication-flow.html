Register vs Login

Key Differences

Register

POST /register
  • Creates a new user
  • Uses RegisterDto
  • Uses UserManager.CreateAsync
  • Stores the user in the database
  • Returns success or validation errors

Login

POST /login
  • Authenticates an existing user
  • Uses LoginDto
  • Uses UserManager.FindByEmailAsync
  • Uses SignInManager.CheckPasswordSignInAsync
  • Generates a JWT Token
  • Returns the token to the client
